# 🍬 CandyBot

**CandyBot** is an automated robotic system designed to **sort, manage, and dispense candy** in a controlled, customizable, and user-friendly way.

The system combines computer vision, inventory management, and interactive interfaces to deliver a seamless experience for both users and operators.

> ⚠️ CandyBot is designed to operate with a **single, specific type of candy** that has uniform size and shape. The exact candy type will be defined in the short future.

---

## 🎯 Project Scope

CandyBot focuses on solving three core problems:

- **Automatic sorting** of candies based on visual characteristics  
- **Controlled dispensing** according to user preferences  
- **Smart inventory management** with minimal human intervention  

The system is designed to be:
- **Autonomous** → operates with minimal supervision  
- **Interactive** → intuitive control via touchscreen and mobile app  
- **Scalable** → adaptable to different candy types and configurations  
- **Robust** → capable of handling edge cases like overflow or low stock  

---

## ⚙️ Core Functionalities

### 1. Candy Sorting

CandyBot automatically classifies candies using computer vision.

- Detects candy properties (e.g., color, shape)  
- Assigns each candy to a corresponding storage compartment  
- Continuously adapts to the configured candy type  

---

### 2. Candy Dispensing

The system delivers candies based on user input.

- Selection by category (e.g., color or type)  
- Configurable quantity  
- Custom mixes (combining multiple categories)  
- Controlled release to ensure consistency  

---

### 3. Touchscreen Interface

CandyBot is operated through an integrated **touchscreen UI**.

- Intuitive navigation  
- Real-time feedback (status, actions, errors)  
- Selection of candy types and quantities  
- System status visualization  

---

### 4. Mobile App Integration

CandyBot can be accessed remotely via a mobile application.

- Monitor inventory levels in real time  
- Receive alerts and notifications  
- Trigger or schedule dispensing  
- Configure system behavior  

---

### 5. Inventory Management

The system continuously tracks candy levels in each compartment.

- Real-time stock monitoring  
- Low stock alerts  
- Refill notifications  
- Usage tracking  

---

### 6. Overflow Handling

CandyBot includes an automatic overflow management system.

- Detects when a storage compartment is full  
- Redirects incoming candy to a dedicated **overflow/reject container**  
- Prevents system blockage and maintains continuous operation  

---

## 🧠 System Behavior

CandyBot operates as an integrated intelligent system:

1. Candies enter the system  
2. They are analyzed and classified  
3. Stored in dedicated compartments  
4. Inventory is updated in real time  
5. Users interact via touchscreen or app  
6. Candies are dispensed on demand  
7. Edge cases (e.g., overflow) are handled automatically
